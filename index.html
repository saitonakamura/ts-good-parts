И что мне выбрать?

Any это
"Серая слизь"

any

  • Отключает все проверки
  • Растекается по кодовой базе
  • Полезен в исключительных случаях

unknown

  • TypeScript 3+
  • Заставляет сделать рантайм проверки
  • Нужен при общении с внешней средой

Strict

strictNullChecks

  • Ошибка на миллиард долларов
  • number | null

noImplicitAny

  • any только явно
  • any меньше растекается
